How Loss Functions Define the "Rules of the Game"

Pixel MSE Loss: Forces exact pixel matching → Identity mapping

Distribution MSE Loss: Matches histogram of values → Allows pixel rearrangement

Add Shape Constraints:

  • Smoothness: Encourages local consistency
  • Direction: Creates left-dark to right-bright gradient

Result: Noise rearranges into structured gradient through backpropagation
